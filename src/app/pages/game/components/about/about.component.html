O objetivo deste projeto é permitir que o Elifoot 98 seja jogado diretamente no navegador, em computadores e celulares, sem precisar de extensões do sistema ou emuladores. Como bônus, esta versão permite jogar em modo janela, algo não disponível no original ou via OTVDM.
Para mais detalhes sobre o projeto, consulte o nosso
repositório oficial no GitHub.

Sobre o Elifoot 98

Elifoot 98 é um jogo original de André Elias, distribuído como shareware.
Este projeto é uma homenagem de fãs, sem fins lucrativos.
