Menu de Opções

O menu de opções (> ou ) localizado no canto superior da tela.

Desativar Suavização
Desativa o filtro de suavização de imagem do emulador. Caso prefira os pixels quadrados, ative essa opção.

Essa opção pode ser útil para quem está jogando em telas de baixa resolução ou dispositivos mais antigos, onde a suavização pode deixar a imagem menos legível.

 Auto-Save

Submenu com opções de salvamento de dados da máquina virtual/emulador.

Salvar Máquina Virtual

Salva o estado atual da máquina virtual, incluindo o sistema de arquivos do emulador em geral. Essa opção é útil caso você tenha feito alterações nos arquivos em geral, como a utilização do editor de equipes do Elifoot 98 ou mudanças nas configurações do windows 3.1 por exemplo.

Desativar Sync Save

Por padrão, os sistema detecta automaticamente quando o jogo é salvo e após o fim do salvamento, dispara o salvamentento automático da máquina virtual/emulador.

Utilize essa opção para desativar o salvamento automático da máquina virtual/emulador.

Save Periódico (5min)

Utilize essa opção para ativar o salvamento automático da máquina virtual/emulador a cada 5 minutos.

Note que essa opção é independente da de Sync Save, ou seja, é possível ativar as duas opções ao mesmo tempo.

Dados e Backups

Submenu com opções para importação e exportação de dados de jogos além de opção para apagar e resetar a máquina.

Exportar Jogos Salvos

Opção para fazer download dos jogos salvos (arquivos .e98) da máquina virtual para seu computador/celular.
Importante: O download é feito em um arquivo compactado (.zip) que contém todos os jogos salvos.

Importar Jogos Salvos

Opção para copiar um jogo salvo em outro local (arquivo .e98) para dentro da máquina virtual/emulador.
Importante: Se já houver um arquivo com mesmo nome dentro da máquina virtual, o save selecionado para importação irá SUBSTITUIR o arquivo no destino.

Apagar todos os dados

Opção para apagar todos os dados da máquina virtual/emulador, incluindo jogos salvos, arquivos de configuração e outros dados.
Importante: Essa opção não pode ser desfeita! Uma vez apagados os dados, não há como recuperá-los.
