Menu de Opções

O menu de opções (> ou ) localizado no canto superior da tela.

Desativar Suavização
Desativa o filtro de suavização de imagem do emulador. Caso prefira os pixels quadrados, ative essa opção.

Essa opção pode ser útil para quem está jogando em telas de baixa resolução ou dispositivos mais antigos, onde a suavização pode deixar a imagem menos legível.

 Auto-Save

Submenu com opções de salvamento de dados da máquina virtual/emulador.

Salvar Máquina Virtual

Salva o estado atual da máquina virtual, incluindo o sistema de arquivos do emulador em geral. Essa opção é útil caso você tenha feito alterações nos arquivos em geral, como a utilização do editor de equipes do Elifoot 98 ou mudanças nas configurações do windows 3.1 por exemplo.

Desativar Sync Save

Por padrão, os sistema detecta automaticamente quando o jogo é salvo e após o fim do salvamento, dispara o salvamentento automático da máquina virtual/emulador.

Utilize essa opção para desativar o salvamento automático da máquina virtual/emulador.

Save Periódico (5min)

Utilize essa opção para ativar o salvamento automático da máquina virtual/emulador a cada 5 minutos.

Note que essa opção é independente da de Sync Save, ou seja, é possível ativar as duas opções ao mesmo tempo.

Dados e Backups

Este submenu oferece opções para gerenciar dados de jogos, incluindo exportação, importação e redefinição completa da máquina virtual.

Exportar Jogos Salvos

Permite baixar os jogos salvos (arquivos .e98) da máquina virtual para o seu dispositivo.
Nota: O download será feito em um arquivo compactado (.zip) contendo todos os jogos salvos.

Importar Jogos Salvos

Permite adicionar um jogo salvo de outro local (arquivo .e98) para dentro da máquina virtual.
Atenção: Caso já exista um arquivo com o mesmo nome na máquina virtual, o arquivo importado irá SUBSTITUIR o existente.

Apagar Todos os Dados

Remove todos os dados da máquina virtual, incluindo jogos salvos, configurações e outros arquivos.
Aviso: Esta ação é irreversível. Após a exclusão, os dados não poderão ser recuperados.

Input de Texto

Permite abrir uma caixa de texto para enviar conteúdo diretamente para a máquina virtual.
Essa funcionalidade é útil para colar textos do seu computador ou celular diretamente no emulador, ou para digitar caracteres que não estão disponíveis no teclado virtual.

Teclado Virtual

Ativa ou desativa o teclado virtual.

 Opções de Patch

Este submenu oferece opções para aplicação de patches para alteração de times, bandeiras, países, árbitros e demais opções dentro do jogo.

Esses patches podem ser encontrados em sites especializados, como os dos nossos amigos do Turbo Elifoot .

Carregar Patch

Permite selecionar um arquivo de patch para aplicar na máquina virtual.
Nota: Apenas arquivos ZIP com estrutura compatível com o Elifoot 98.

Remover Patch

Remove o patch atualmente aplicado na máquina virtual e restaura jogo original.
Atenção: Jogos salvos não serão apagados, mas podem ficar em estado corrompido até que o patch seja reaplicado.
