## Desenvolvimento
### Requisitos
- Node.js 16 ou superior

### Rodando localmente
1. `npm install` - Na primeira vez que for rodar o projeto, caso já tenha feito isso, pule este passo.
2. `npm start`
3. Pronto, abra o navegador e acesse http://localhost:4200
